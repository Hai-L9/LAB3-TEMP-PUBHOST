This is an MMVC powered AI voice changer that ran in real time. It lets users use pre-trained ai voice models to convert their speech in real time. All software used is not owned by me I only created a UI and merged mutliple models to allow for realtime converstion.

VRC-DMX is a collection of shaders, scripts, and models designed to emulate professional stage lighting into VRChat with Unity 2019.4 and UdonSharp.

  • Translates input from industry standard (Artnet) DMX information to control virtual lightsources
  • Handles synchronization of game enviornments for all remote players for live performances
  • Drag-and-drop for VRChat and Unity world creation with a simple UI to allow Stage-techs and DJs with little coding experience to easily learn and use effectively.
